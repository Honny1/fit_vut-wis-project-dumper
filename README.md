# FIT VUT WIS Project Dumper

Requirements:
- `Python >=3.9`
- (Optional) `pipenv`

### Installation
```
pipenv install
```

or

```sh
pip3 install -r requirements.txt
```

### Usage
```
pipenv run python main.py [-v[v]] [-o|--output OUTPUT_DIR]
```

or

```sh
python3.9 main.py [-v[v]] [-o|--output OUTPUT_DIR]
```
